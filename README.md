# RancheAR — Releases

Este repositorio contiene los instaladores oficiales de **RancheAR**, la plataforma de juegos de fiesta para jugar con amigos.

## Descargar

Andá a la sección de [**Releases**](../../releases) y bajate el instalador para tu sistema operativo:

| Sistema | Archivo |
|---------|---------|
| Windows | `.exe` |
| macOS | `.dmg` |
| Linux | `.AppImage` |

## ¿Qué es RancheAR?

Una plataforma de juegos de fiesta para jugar en grupo, inspirada en Jackbox, hecha por y para argentinos.

- El **host** descarga e instala la app en su PC
- Los **jugadores** se unen desde cualquier dispositivo con un navegador — sin instalar nada
- Solo necesitás un código de sala para entrar

---

*El código fuente se encuentra en un repositorio privado. Este repo es solo para distribución de instaladores.*
